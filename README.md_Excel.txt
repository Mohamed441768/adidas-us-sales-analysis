# adidas-us-sales-analysis

**Excel dataset and analysis of Adidas US sales, including KPIs, trends, and regional performance insights.**

---

## 📊 Project Overview

This project contains an Excel dataset that represents sales data for Adidas products in the US market. It can be used for:

- Building dashboards
- Performing data analysis
- Practicing Excel or Power BI skills

The dataset includes:

- Order date
- Retailer and region
- Product category
- Units sold
- Total sales
- Operating profit

---

## 🔍 Insights You Can Discover

- 🏆 Best-selling products
- 🌍 Sales by region
- 📈 Revenue growth over time
- 💰 Profitability trends

---

## 🛠️ Tools You Can Use

- Microsoft Excel
- Power BI
- Python (Pandas, Matplotlib, Seaborn)
- Google Sheets (optional)
